Capstone Project

MANAGEMENT INFORMATION SYSTEM

Office of the Senior Citizens Affairs (OSCA) System

As part of my capstone, my team and I developed a comprehensive Management Information System specifically for the Office of the Senior Citizen Affairs (OSCA) in Panabo City.

The system was built using JavaScript, Node.js, and PostgreSQL, featuring financial monitoring, senior citizens database management, report generation, and automated applicant actions. It replaces manual paper tracking to make OSCA services significantly faster and more accurate.

System Architecture & Tech Stack



  • ✔ Frontend: JavaScript for dynamic and interactive user interface development
  • ✔ Backend: Node.js for server-side logic and RESTful API handling
  • ✔ Database: PostgreSQL for structured data storage and efficient query management
